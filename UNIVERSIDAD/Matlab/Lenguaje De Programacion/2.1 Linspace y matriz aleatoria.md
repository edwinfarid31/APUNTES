## Linspace
Linspace sirve para crear un arreglo espaciado en el cual se conoce el primer, ultimo elemento y el numero de elementos entre ellos.
sintaxis: 
		 La sintaxis de linspace es la siguiente.
		 *linspace(primero, ultimo, numero de elementos)*

```matlab 
x= linspace(0,1,5)
% x = 0 0.250 0.500 0.750 1
```

#### características 
- Es vital separar las entradas de la función linspace con comas.
- linspace crea un vector fila.

para volver un vector fila en un vector columna se usa $`$  arriba de la  variable que contiene el arreglo.
```matlab 
x = [1 2 3];
% volverlo un vector columna 
x`
%{ salida resultado 
1
2
3
}%
```
