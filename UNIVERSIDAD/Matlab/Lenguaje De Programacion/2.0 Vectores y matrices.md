#MATLAB 

### Introducción 
En MATLAB todas las variables son arreglos por lo que una variable puede contener matrices de datos sin necesidad de definir extras. 

### Vectores y matrices
#### Vector 
Cuando en MATLAB separamos los medio de comas o espacio MATLAB lo toma como un vector.

A continuación se muestra un ejemplo de este concepto. 
```matlab
vector1 = [1 2 3]; %ejemplo de vector definido por espacios
vector2 = [1,2,3]; %ejemplo de un vector definido por comas
```

> Los vectores anteriores producen el mismo resultado en consola lo que cambia es la forma en la que se definen. 

#### Matriz 
Cuando en MATLAB separamos los elementos con un **==;== "punto y coma"** MATLAB lo toma como una columna y esta es la clave de la creación de matrices. 

```matlab
matriz = [1 2 3; 4 5 6; 7 8 9];%codigo de una matriz 3x3
```

#### Vectores espaciados 
Existe una forma de crear vectores espaciados en MATLAB y  un vector espaciado es una secuencia de números ordenada y al igual que en múltiples lenguajes existe el paso  y la estructura para crear un vector espaciado es la siguiente 

> [!NOTE] Estructura 
> **variable = inicio:paso:fin** 
> Cuando no se define el paso se toma como paso uno automáticamente 

**Ejemplo**
```matlab
secuencia = 1:5; %secuecnia de paso uno
secuecnia2 = 1:2:10; %seceucnia de paso dos
```
