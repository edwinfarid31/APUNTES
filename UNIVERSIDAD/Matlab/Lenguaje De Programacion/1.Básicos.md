### Constantes

```matlab

pi % constante ya incorporada  en MATLAB
```

### Espacio de trabajo 
En MATLAB existe algo llamado espacio de trabajo el cual es un archivo ==.mat== en el cual se guardan variables del entorno 
#### ¿Cómo se crea?
Para crear un espacio de trabajo usamos el siguiente comando.
````matlab
save datafile.mat 
`````


> [!NOTE] Importante 
> con ese comando guardamos todo el espacio de trabajo en el archivo que lleva por nombre datafile, es importante saber que datafile puede ser cualquier nombre.
**El archivo .mat se guarda en la misma carpeta del entorno en el que se se esta trabajando.**


#### ¿Cómo se carga el archivo?
Para cargar el entorno se usa el siguiente comando. 
```matlab
load datafile.mat
```

> [!NOTE] Importante
> Con este comando cargamos en el entorno las variables guardadas en el archivo datalife.mat
